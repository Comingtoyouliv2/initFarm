# initFarm

AI-powered DeFi robo-advisor on Initia. Verifies on-chain APY, scores protocol risk, builds optimized portfolios, and executes via wallet in one click.

## How It Works

Users define goals through conversational IPS flow. The AI verifies APY using on-chain data, scores risk (TVL stability, audit history, IL exposure, liquidity depth), and constructs portfolios (conservative/balanced/aggressive). Execute allocations directly via the vault contract.

## Initia Integration

- **InterwovenKit** wallet connection (Privy, MetaMask, Keplr)
- **Auto-Sign Sessions** for MsgSend and MsgExecute
- **.init Username** support
- Per-chat INIT micro-payments with on-chain tx receipts
